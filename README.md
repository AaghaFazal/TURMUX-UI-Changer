#### Installation Commands


1.  apt update && apt upgrade

2.  apt install git -y

3.  cd
    git clone https://github.com/lgcyalex/TURMUX-UI-Changer

4.  cd TURMUX-UI-Changer

5.  turmux +x setup.sh

6.  bash setup.sh

7.  exit -y

   
   
   
       **Done ✅**
